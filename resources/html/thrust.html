Thrust Vector Control

Thrust Control

TVC, as we call it, is the action of directing the thrust of an engine's output. Using this we hope to soon be able to fly a model rocket without fins! We are using the ZETA Gen 3 (ZG3) flight controller to power the rocket. It uses the PID algorithm to control and direct the rocket. This technology will beused in most future projects. It is still in its experimental phase.
